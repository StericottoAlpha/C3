# C3 ステリコットα ドキュメント

## システム概要

**C3 ステリコットα**は、複数店舗を展開するレストラン企業向けの日報登録・分析システムです。PDCAサイクルを支援し、店舗運営の改善を促進します。

### 主な機能

| 機能 | 説明 |
|------|------|
| **日報管理** | 店舗スタッフが日々の出来事（クレーム、賞賛、事故、報告など）を記録 |
| **掲示板（BBS）** | 日報を全社で共有し、ナレッジの蓄積とコミュニケーションを促進 |
| **分析ダッシュボード** | 売上、客数、違算金額などのKPIをグラフで可視化 |
| **カレンダービュー** | 日別の日報・実績を確認 |
| **AIチャット** | RAGを活用したAIアシスタントが過去の日報やマニュアルから情報を検索・回答 |
| **月次目標管理** | 店舗ごとの目標設定と達成率の管理 |

### 技術スタック

- **バックエンド**: Django 5.2.4 / Python 3.11
- **データベース**: PostgreSQL 17（pgvector拡張対応）
- **フロントエンド**: HTMX + Django Templates
- **AI**: LangChain + OpenAI GPT-4o-mini
- **ストリーミング**: FastAPI + Uvicorn（チャット用）
- **ストレージ**: Supabase Storage（S3互換）
- **デプロイ**: Render

## クイックリファレンス

### アプリケーション構成

```
c3_app/          # プロジェクト設定
accounts/        # ユーザー認証・管理
stores/          # 店舗・月次目標管理
reports/         # 日報管理
bbs/             # 掲示板
analytics/       # 分析ダッシュボード
ai_features/     # AIチャット・RAG
common/          # 共通機能（ホーム、PWA）
```

### 主要URL

| パス | 機能 |
|------|------|
| `/` | ホーム画面 |
| `/accounts/login/` | ログイン |
| `/report/register/` | 日報作成 |
| `/report/list/` | 日報一覧 |
| `/bbs/list/` | 掲示板一覧 |
| `/analysis/` | 分析ダッシュボード |
| `/ai/chat/` | AIチャット |
| `/admin/` | Django管理画面 |

### ユーザー種別

| 種別 | 権限 |
|------|------|
| `admin` | 管理者 - 全機能にアクセス可能 |
| `manager` | 店長 - 自店舗の管理機能にアクセス可能 |
| `staff` | スタッフ - 日報登録・閲覧が可能 |
